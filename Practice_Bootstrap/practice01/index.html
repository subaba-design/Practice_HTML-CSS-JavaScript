ファイルをダウンロードして使う方法

  1. 公式サイトからファイルをダウンロード
  2. 解凍したファイルからcssファイルとjsファイルをプロジェクトのディレクトリに入れる
    • bootstrap.min.css
    • bootstrap.min.js
  3. jQueryをダウンロードする（BootstrapはjQueryに依存している）
    • 「リンク先を別名で保存」
  4. HTMLにファイルを読み込む
    • headタグに「link:css」
    • bodyの末尾に「script:src」
      1. jquery（こっちを先に読み込む！）
      2. bootstrap
  5. 公式サイトからコピペ
