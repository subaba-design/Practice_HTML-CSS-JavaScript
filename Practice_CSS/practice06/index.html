テキストのスタイル変更方法で学んだこと

2018年10月16日

テキストの大きさを変更する

テキストの大きさを変更するにはfont-sizeプロパティを使います。font-sizeの使い方は次のとおりです。

p { font-size: 12px; }

上記の例では「px」を使いましたが、その他にも「%」「em」「rem」など様々な単位を利用することが出来ます。

テキストの太さを変更する

テキストの太さを変更するにはfont-weightプロパティを使います。font-sizeの使い方は次のとおりです。

p { font-weight: bold; }

上記の例では「bold」を使いましたが、その他にも「normal」「lighter」「bolder」「数値指定」など様々な単位を利用することが出来ます。

flexboxで学んだこと

2018年10月17日

flexbox

flexboxとはフレックスコンテナーとフレックスアイテムを使って、表示要素を縦並び、横並び、表示順番を逆にすることが可能になる機能のことである。

フレックスコンテナーとはフレックスアイテムを格納するコンテナーのことで、フレックスコンテナーにしたい要素のスタイル定義に以下のように「display: flex;」をセットすることで利用できる。

div { display: flex; }

フレックスアイテムとはフレックスコンテナーに格納された実際に表示される要素となる。
